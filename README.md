Stable utility apps are maintained inside [LocalFirstApps](https://shfqrkhn.github.io/LocalFirstApps/) instead of promoted as separate standalone surfaces.

## Project Principles

- Keep tools simple enough to run from GitHub Pages when possible.
- Prefer local-first, privacy-conscious workflows where a backend is not needed.
- Keep maintenance focused on the few projects that are most useful.
- Make assumptions, data, and limitations visible instead of hiding them.
